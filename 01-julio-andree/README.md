## Los 4 pilares del pensamiento computacional  
**1.- descompocision:** Un problema de descompone en diferentes grupos.  
**2.- Reconiciminto de patrones:** Encontrar similitudes/tendencias dentro de un problema, si algo se repite podemos automatizarlo.  
**3.- Abstracción:** Lo importante vs el detalles, filtar la info innecesario y quedarse solo con las caracteristicas que definen el problema.  
**4.- Los algoritmos:** La receta paso a paso, serie de reglas ordenadas para resolver un problema, en el codigo se traduce como el diagraa de flujo y en las condicionales.
